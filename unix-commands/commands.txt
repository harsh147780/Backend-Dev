================================================================
                        UNIX COMMANDS
================================================================

--- LISTING & NAVIGATION ---
ls                          Lists files/dirs in current location
ls -l                       Detailed list (permissions, size, owner)
ls -a                       Shows hidden files
cd [directory]              Changes the current directory
cd ..                       Moves up one level
cd ~                        Moves to the home directory
pwd                         Prints full path of current working directory

--- CREATION & DELETION ---
mkdir [name]                Creates a new directory
rmdir [name]                Removes an empty directory
touch [file]                Creates empty file or updates timestamp
rm [file]                   Deletes a file
rm -r [directory]           Deletes a directory and contents recursively
rm -f                       Force delete without confirmation

--- MANIPULATION & VIEWING ---
cp [source] [dest]          Copies files or directories
cp -r                       Copies directories recursively
file [file]                 Determines the file type
cat [file]                  Displays the entire content of a file
